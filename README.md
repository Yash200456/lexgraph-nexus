# LexGraph Nexus

A cloud-powered GraphRAG system that analyzes legal documents to detect and explain hidden contradictions.

## 🎯 Project Overview

LexGraph Nexus uses AI and graph databases to help legal teams quickly identify conflicts in contracts.

### Key Features
- 📄 PDF text extraction with intelligent chunking
- 🤖 AI-powered entity and relationship extraction
- 🔗 Graph database for visualizing legal connections
- ⚡ Cloud-based architecture (works on any hardware)
- 🎨 Interactive React UI with graph visualization

## 🛠️ Tech Stack

**Backend:** Python, LangChain, Neo4j AuraDB, FastAPI
**Frontend:** React, Tailwind CSS, React Force Graph
**AI/ML:** OpenAI GPT-4o-mini / Google Gemini, Embeddings

## 📦 Installation

1. Clone the repository
2. Create virtual environment: python -m venv venv
3. Activate: .\venv\Scripts\activate (Windows)
4. Install: pip install -r requirements.txt
