# LexGraph Nexus

A cloud-powered GraphRAG system that analyzes legal documents to detect and explain hidden contradictions across multiple files.

## 🎯 Project Overview

LexGraph Nexus uses AI and graph databases to help legal teams quickly identify conflicts and inconsistencies in contract documents. The system extracts entities from legal PDFs, builds a knowledge graph of relationships, and uses retrieval-augmented generation to answer questions about potential contradictions.

## ✨ Key Features

- 📄 PDF text extraction with intelligent chunking
- 🤖 AI-powered entity and relationship extraction
- 🔗 Graph database for visualizing legal connections
- ⚡ Cloud-based architecture (works on any hardware)
- 🎨 Interactive React UI with graph visualization
- 🔍 Natural language query interface for contract analysis

## 🛠️ Tech Stack

**Backend:** Python, LangChain, Neo4j AuraDB, FastAPI  
**Frontend:** React, Tailwind CSS, React Force Graph  
**AI/ML:** OpenAI GPT-4o-mini / Google Gemini 1.5 Flash, text-embedding-3-small
